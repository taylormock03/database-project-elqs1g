Year 11 Example Project Template 2123

This template holds the names of people in a database. He demonstrates the concepts of Angular, breaking your app into components and the use of local storage and the concept of CRUD (Create, Read, Update and Delete). The components here include
- Add form (data to localstorage)
- List of data (put data fron localstorage)
- Home page
- Services (contains functions that many components need to access - eg AddPerson)
- Dialog ( has a dialogue box that is used when you want to delete a record)
- horizontal-menu (which lists the menu items)
